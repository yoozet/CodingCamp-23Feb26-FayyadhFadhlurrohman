# Portofolio Pribadi 🚀

Halo! Ini adalah repository buat source code website portofolio pertama yang aku buat. Project ini aku kerjakan buat latihan nerapin materi web development kayak HTML, CSS, JavaScript, dan framework Tailwind CSS.

Bisa langsung cek live preview-nya di sini ya:  
🔗 **[Live Preview Portofolio](https://yoozet.github.io/CodingCamp-23Feb26-FayyadhFadhlurrohman/)**

---

## 💻 Tech Stack

- **HTML & CSS**: Buat kerangka dasar sama animasi kustom (kayak efek blob di background).
- **Tailwind CSS**: Biar gampang dan rapi ngatur layout responsif (bisa enak diliat lewat HP atau laptop).
- **JavaScript (Vanilla)**: Buat nanganin logika simpel, contohnya animasi scroll, highlight menu navbar pas scroll, sama validasi form kontak.

## � Isi File

- `index.html` → Halaman utamanya.
- `style.css` → Styling tambahan di luar tailwind.
- `script.js` → Buat interaksi webnya.
- Folder `assets/` → Isinya foto profil, icon favicon, sama screenshot project.
